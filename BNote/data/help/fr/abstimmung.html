Il est fréquent que des dates, des pièces, des programmes, des lieux, etc. doivent être concertés. Il peut y avoir une ou plusieurs possibilités et tout le monde n'est pas toujours autorisé à voter. Pour effectuer ces votes, il est possible d'utiliser le module de vote.

Ouverture du module

Dans la mesure où l'utilisateur a l'autorisation pour le module Vote, il peut l'ouvrir via le menu principal. Un aperçu de tous les votes apparaît d'abord, ainsi qu'un formulaire pour créer de nouveaux votes.

Création d'un vote

Pour créer un nouveau vote, remplissez le titre et la date de fin du vote. D'autres options sont disponibles :

  • Vote par date: S'agit-il d'un vote avec un ou des dates ? Si oui, coche la case "Vote par date".
  • Choix multiple possibles: Un seul choix est-il possible pour le vote ou est-il possible de donner plusieurs réponses ? Si plusieurs possibilités sont offertes, la case doit être remplie.

Si l'on clique sur OK, une fois le vote créé avec succès, une page s'affiche avec 2 boutons : Ajouter des options et Ajouter des participants au vote. Les deux peuvent être modifiés ultérieurement.
Chaque vote requiert des options et des votants.

Options

Sur l'écran Options, un champ "Ajouter une option" se trouve sous les options déjà créées. On y saisit le nom de l'option et on confirme avec OK. L'option est alors ajoutée au vote. Pour supprimer une option, il suffit de cliquer sur l'option. Les options actuelles sont affichées au-dessus du champ Ajouter une option.

Personnes habilitées à voter

Sur l'écran des personnes autorisées à voter, sous l'auteur qui est ajouté par défaut au vote, se trouve un bouton Ajouter des membres et un champ Ajouter des personnes habilitées à voter. Si l'on clique sur le bouton, tous les membres sont ajoutés au vote. Les membres individuels peuvent être ajoutés via le champ en bas. Pour supprimer une personne autorisée à voter, il suffit de cliquer sur celle-ci.

Afficher les votes

Sur la page d'accueil de chaque module, tous les votes sont affichés. Si l'auteur d'un vote clique dessus, les options suivantes s'affichent :

  • Modifier le vote : Modifier les propriétés du vote.
  • Supprimer le vote : Supprime le vote.
  • Options : Édition des options de vote - voir ci-dessus.
  • Votants : Qui peut participer au vote ?
  • Résultat : Le résultat du vote
  • Retour : Retour à la page d'accueil

Chaque utilisateur peut voir tous les votes, mais ne peut pas les modifier. Seul l'auteur du vote peut le modifier. Tous peuvent consulter le résultat du vote.

Résultat du vote

Le résultat du vote est affiché pour l'auteur du vote via le bouton Résultat pour tous les autres utilisateurs, en cliquant sur le vote. Le résultat est toujours actualisé, ce qui permet d'afficher des résultats intermédiaires avant la fin du vote.
