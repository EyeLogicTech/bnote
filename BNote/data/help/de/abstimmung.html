Häufig müssen Termine, Stücke, Programme, Orte, etc. abgestimmt werden. Es kann eine oder mehrere Optionen geben und nicht immer sind alle Abstimmungsberechtigt. Um diese Abstimmungen durchzuführen, kann das Abtimmung-Modul verwendet werden.

Modul öffnen

Sofern der Benutzer die Berechtigung für das Modul Abstimmung hat, kann er dieses über das Hauptmenü aufrufen. Zunächst erscheint eine Übersicht aller Abstimmungen und ein Formular zur Erstellung neuer Abstimmungen.

Abstimmung Erstellen

Um eine neue Abstimmung zu erstellen, fülle den Titel und das Abstimmungsende aus. Weitere Optionen sind:

  • Datumgsabstimmung: Handelt es sich um eine Datumsabstimmung? Wenn ja, dann klick das Kästchen hinter "Datumsabstimmung" an.
  • Mehere Optionen möglich: Ist nur eine der Optionen für die Abstimmung möglich oder können Stimmen für mehrere Optionen vergeben werden? Wenn mehrere Optionen möglich sind, muss das Kästchen ausgefüllt werden.

Klickt man auf OK, wird nach erfolgreichem Erstellen der Abstimmung eine Seite angezeigt mit 2 Buttons: Optionen hinzufügen und Stimmberechtigte hinzufügen. Beides kann nachträglich geändert werden.
Jede Abstimmung benötigt Optionen und Abstimmungsberechtigte.

Optionen

Auf dem Bildschirm Optionen befindet sich unter den bereits angelegten Optionen ein Feld "Option hinzufügen". Hier wird der Name der Option eingegeben und mit OK bestätigt. Die Option wird der Abstimmung dann hinzugefügt. Um eine Option zu löschen, klickt man auf die Option. Über dem Option hinzufügen Feld werden die aktuellen Optionen angezeigt.

Abstimmungsberechtigte

Auf dem Bildschirm Abstimmungsberechtigte befindet sich unter dem Autor, der standardmäßig der Abstimmung hinzugefügt wird, ein Button Alle Mitglieder hinzufügen und ein Feld Abstimmungsberechtigte hinzufügen. Klickt man auf den Button, so werden alle Mitglieder der Abstimmung hinzugefügt. Einzelne Mitglieder können über das Feld unten hinzugefügt werden. Um einen Abstimmungsberechtigten wieder zu löschen, klickt man auf diesen.

Abstimmungen anzeigen

Auf der Startseite jedes Moduls werden alle Abstimmungen angezeigt. Klickt der Autor einer Abstimmung auf diese, so werden ihm folgende Optionen angezeigt:

  • Abstimmung bearbeiten: Eigenschaften der Abstimmung bearbeiten.
  • Abstimmung löschen: Löscht die Abstimmung.
  • Optionen: Bearbeitung der Abstimmungsoptionen - siehe oben.
  • Abstimmungsberechtigte: Wer darf an der Abstimmung teilnehmen?
  • Ergebnis: das Ergebnis der Abstimmung
  • Zurück: Zurück zur Startseite

Jeder Nutzer kann alle Abstimmungen einsehen, jedoch nicht bearbeiten. Nur der Autor der Abstimmung kann diese bearbeiten. Alle können das Ergebnis der Abstimmung anzeigen.

Abstimmungsergebnis

Das Abstimmungsergebnis wird für den Autor der Abstimmung über den Button Ergebnis angezeigt, für alle anderen Nutzer über einen Klick auf die Abstimmung. Das Ergebnis ist immer aktuell, so können also auch Zwischenergebnisse vor Beendigung der Abstimmung angezeigt werden.
