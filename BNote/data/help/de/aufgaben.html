Mit diesem Modul können Aufgaben erstellt und einem Benutzer zugewiesen werden. Ist eine Aufgabe erledigt, so kann der Ersteller dies zur Kenntnis nehmen und die Aufgabe löschen.

Rollen

Um eine Aufgabe zu erstellen benötigt der Benutzer Zugriff auf das Aufgaben Modul. Die Zugriffssteuerung befindet sich im User-Modul unter dem Link "Rechte bearbeiten" in den Benutzerdetails. Der Zugriff auf das Modul erfolgt über die Hauptnavigation links unter "Aufgaben".

Im Zusammenhang mit Aufgaben werden zwei Nutzer unterschieden: der Aufgabensteller und der Verantwortliche. Der Aufgabensteller erstellt eine Aufgabe und löscht diese mit Hilfe des Aufgabenmoduls. Der Verantwortliche sieht die Aufgabe auf seiner Startseite und kann diese als erledigt markieren.

Aufgabenersteller

Aufgabe erstellen

Der Aufgabenersteller öffnet das Aufgabenmodul und klickt auf "Aufgabe hinzufügen". Es erscheint ein Formular mit einem Aufgabentitel und eine genaueren Beschreibung der Aufgabe. Außerdem sollte die Fälligkeit der Aufgabe sowie ein Verantwortlicher eingetragen werden. Mit "OK" wird die Aufgabe erstellt.

Gruppenaufgabe erstellen

Eine Gruppenaufgabe ist vergleichbar mit einzelnen Aufgaben, die man für jedes Gruppenmitglied anlegt. Somit ist es einfach möglich zu prüfen, wer die Aufgabe bereits erledigt hat und wer nicht. Das Hinzufügen einer Gruppenaufgabe ist wie eine einfache Aufgabe, jedoch über die Schaltfläche "Gruppenaufgabe hinzufügen" und der Auswahl der Gruppe.

Aufgabe einsehen

Öffnet man das Aufgabenmodul erscheinen alle offenen Aufgaben in einer Tabelle. Im Details eine Aufgabe anzuzeigen klickt man auf eine Zeile. Es erscheint die Detailansicht einer Aufgabe mit weiteren Informationen und Optionen.

Aufgabe als abgeschlossen markieren

In der Detailansicht einer Aufgabe gibt es die Option "Als abgeschlossen markieren". Drückt man auf diesen Knopf so wird die Aufgabe als abgeschlossen markiert und ist damit für den Verantwortlichen nicht mehr sichtbar.

Offene/Abgeschlossene Aufgaben anzeigen

Zur Einsicht der abgeschlossenen Aufgaben kann man auf der Modulstartseite auf den Knopf "Abgeschlossene Aufgaben anzeigen" klicken. Damit ändert sich das Inhalt der darunterliegenden Tabelle in alle abgeschlossenen Aufgaben.

Aufgabe löschen

In der Detailansicht einer Aufgabe gibt es die Option "Aufgabe löschen". Klickt man auf sie wird man gefragt ob man sicher ist, diese löschen zu wollen. Bestätigt man dies wird die Aufgabe löscht.

Verantwortlicher

Aufgabe einsehen

Jedem Benutzer werden seine Aufgaben (die Aufgaben bei denen er als Verantwortlicher eingetragen ist) auf der Startseite unten links angezeigt. Sind keine Aufgaben offen, so erscheint "Keine Aufgaben vorhanden". Ist eine Aufgabe vorhanden kann durch Klick auf den Aufgabentitel Details zur Aufgabe angezeigt werden.

Aufgabe als abgeschlossen markieren

Aufgaben können als abschlossen markiert werden indem unter der Aufgabe auf den Link "als abgeschlossen markieren" geklickt wird. Danach verschwindet die Aufgabe aus der Ansicht und sie wird beim Aufgabensteller als abgeschlossen angezeigt.
